# Photos

Document your HVAC conversion journey with photos.

## Folder Structure

### /before
Photos of the original HVAC setup:
- Old oil burner/boiler
- Original propane equipment
- Existing ductwork/radiators
- Utility meter photos

### /installation
Progress photos during installation:
- Conduit installation (6" from main house to garage)
- PEX piping runs
- Heat pump placement
- Indoor pumping station setup
- AHU installations

### /equipment
Close-up equipment photos:
- Apollo 5-ton heat pump unit
- Indoor pumping station
- HBX snow melt controller
- Mixing valves
- Circulation pumps
- ESP32 BTU meters

### /completed
Final installation photos:
- Completed system overview
- Each zone finished
- Control panels
- Insulated piping (lesson learned!)

## Photo Tips

- Include a reference for scale when possible
- Take before AND after photos from the same angle
- Document any problems/challenges you encounter
- Label photos with dates and descriptions
