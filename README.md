# ESP32 BTU Meter - HVAC Monitoring System

A DIY HVAC monitoring system using ESP32 microcontrollers and BTU meters for tracking heating and cooling energy usage.

## Features

- Real-time BTU measurement for HVAC zones
- ESP32-based wireless monitoring
- Temperature and flow rate sensing
- Energy usage tracking and logging

## Hardware Components

- ESP32 Development Board
- Temperature sensors (DS18B20 or similar)
- Flow meters
- Power supply

## Project Structure

```
ESP32-BTU-METER/
├── src/           # Main source code
├── lib/           # Libraries and dependencies
├── docs/          # Documentation
├── schematics/    # Wiring diagrams and schematics
└── README.md
```

## Getting Started

1. Clone this repository
2. Install PlatformIO or Arduino IDE
3. Configure your WiFi credentials
4. Upload to your ESP32
